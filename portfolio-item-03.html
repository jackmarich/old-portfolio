C++: Tic Tac Toe Game

Basic game using command prompt

C++ is great for game development, so I started with the simplest game I know - Tic Tac Toe. This is a light game that is played in the console.

It first asks you to enter a spot to play, and then the computer uses the random function to play you back.

You continue back and forth until you find a winner, or you tie!

C++ was the most difficult language for me to learn, so seeing progress into making this application motives me to do more. I hoep that I can start making games in Unreal once I have the time, or with friends.
